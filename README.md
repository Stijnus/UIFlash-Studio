# UIFlash Studio

A powerful AI-driven UI prototyping tool that leverages Gemini 3 Flash to rapidly generate, iterate, and analyze user interfaces.

## 🚀 Features

- **Instant UI Generation**: Describe your UI in natural language and watch it come to life in seconds.
- **Device-Aware Design**: The AI detects if you are in Mobile or Desktop mode and optimizes the generated UI specifically for that device.
- **Image-to-UI (Vision)**: Upload screenshots or sketches to use as a reference for your generated designs.
- **Design Analysis**: Get professional AI feedback on your designs, covering layout, typography, accessibility, and color harmony.
- **Style Variations**: Generate multiple versions of your UI simultaneously using curated "Variation Packs" (e.g., Modern SaaS, Brutalist, Glassmorphism).
- **Responsive Previews**: Toggle between Desktop and Mobile views to ensure your designs look great on all devices.
- **Code Export**: Instantly copy or download the generated HTML/Tailwind CSS code.
- **Image Export**: Capture high-quality PNG screenshots of your designs, automatically sized for the selected device.

## 🛠️ Technologies Used

- **React**: Frontend framework.
- **Tailwind CSS**: Utility-first styling.
- **Gemini 3 Flash**: The engine behind UI generation and design analysis.
- **Lucide React**: Beautiful, consistent iconography.
- **Motion (Framer Motion)**: Smooth animations and transitions.
- **html2canvas**: For high-quality design exports.

## 📖 How to Use

1. **Describe your UI**: Enter a prompt in the text area (e.g., "A sleek dashboard for a fitness app").
2. **(Optional) Add Images**: Drag and drop reference images to guide the AI.
3. **Generate**: Click "Generate UI" to create a single design, or "Generate 3 Variations" to see different aesthetic takes.
4. **Iterate**: Use the "Refine" tab to ask for specific changes to your generated design.
5. **Analyze**: Click "Analyze Image" to get detailed feedback on your current design or uploaded references.
6. **Export**: Use the action buttons in the preview header to copy code, download HTML, or save as an image.

## 🎨 Variation Packs

- **Modern SaaS**: Clean, professional, and accessible.
- **Creative & Brutalist**: Bold typography and high-contrast layouts.
- **Glassmorphism**: Soft blurs and translucent surfaces.
- **Minimalist**: Focus on whitespace and essential elements.
